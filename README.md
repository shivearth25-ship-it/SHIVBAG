# SHIV-LUXURY-BAG

Premium wholesale B2B packaging and carry bag catalog storefront for **Shiv Enterprise**.

## Tech Stack
- **Structure**: HTML5
- **Styling**: Vanilla CSS (Obsidian dark & light theme aesthetics, smooth transitions)
- **Logic**: Vanilla JavaScript (Product grid renderer, WhatsApp & Email wholesale enquiry link builders)

## Folder Structure
- `index.html` — Storefront structure
- `style.css` — Styling system
- `app.js` — Product catalog dataset and dynamic link sync
- `*.png` — Clean product images (watermarks removed)

## Local Setup
1. Clone this repository.
2. Double-click `index.html` to open the website locally in any browser.
